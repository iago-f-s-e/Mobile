# Clima
É um aplicativo que mostra a previsão do tempo da localização atual do usuário, mas também mostra de localizações especificadas pelo usuário na tela de "busca".

Apesar de ser um projeto relativamente simples, me trouxe bastante aprendizado. As funcionalidades que mais me agregaram conhecimento, foi o uso de API externa, solicitação de permissões para usuários android e o uso de geolocalização. 


# Gerador de senhas 
Gera senhas aleatórias de forma bem simples. O aplicativo pede o tamanho da senha no limite de 8 a 20 caracteres, após informar o tamanho da senha, basta clicar no botão de gerar.
